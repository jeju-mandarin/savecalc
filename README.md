# 💰 예적금 만능 이자 계산기

은행 예금 및 적금의 만기 수령액을 간편하게 계산해 볼 수 있는 정적 웹 애플리케이션입니다.

🔗 **[지금 바로 사용해보기 (Live Demo)](https://jeju-mandarin.github.io/Bokri)**
*(위 링크의 `Bokri` 부분은 실제 생성하신 저장소 이름으로 변경해주세요.)*

---

## ✨ 주요 기능

이 계산기는 15.4%의 이자 소득세(이자소득세 14% + 지방소득세 1.4%)를 기준으로 **세후 만기 총액**을 계산합니다.

* **4-in-1 계산:**
    * 복리 적금 (월 납입)
    * 단리 적금 (월 납입)
    * 복리 예금 (목돈 예치)
    * 단리 예금 (목돈 예치)
* **세부 항목 표시:**
    * 총 원금
    * 총 이자 (세전)
    * 만기 총액 (세전)
    * 이자 소득세 (15.4%)
    * **만기 총액 (세후)**
* **손쉬운 비교 기능:**
    * '단리' 계산 시 '복리'의 세후 총액을 함께 표시합니다.
    * '복리' 계산 시 '단리'의 세후 총액을 함께 표시합니다.
    * 두 방식 간의 **세후 차액 (얼마나 많음/적음)**을 한눈에 비교할 수 있습니다.
* **동적 UI:** '적금'과 '예금' 선택에 따라 '월 납입액' 또는 '총 예치금' 입력창이 자동으로 표시됩니다.

> **계산 방식 안내**
> *   **복리 적금**: 매월 발생한 이자가 원금에 더해져 다음 달 이자 계산의 기준이 되는 **월 복리** 방식을 시뮬레이션하여 실제 수령액에 가깝게 계산합니다.
> *   **단리 적금**: 매월 납입한 원금에 대해서만 계약 당시의 이율을 적용하여 이자를 계산합니다.
> *   **세금 계산**: 만기 시 발생한 총 이자에 대해 15.4%를 일괄 과세하여 세후 수령액을 계산합니다.
